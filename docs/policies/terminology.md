# Terminology Policy

## Banned Terms

- **syncer** — Do not use this word anywhere in the project: code, docs,
  comments, commit messages, or generated artifacts. Use a precise term
  instead. For the `ContextSync` class, refer to it by name or describe
  it as "the context-sync library", "the `ContextSync` entry point", or
  simply "context-sync" depending on context. When a category label is
  needed (for example in generated reference docs), use "Core" or
  "Entry Point" instead of "Syncer".
