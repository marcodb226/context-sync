Identified bridge import workaround in [src/context_sync/_real_gateway.py](../../src/context_sync/_real_gateway.py): `from linear_client.types import IssueId as UpstreamIssueId` (lines 312, 467) and `IssueKey as UpstreamIssueKey` (line 313) — these exist because Pyright treats the two NewType definitions as distinct types.

The package continues to be importable without `linear-client` installed.
