## Completion Summary

**What was produced:** The public CLI surface is reduced from five commands to
four (`sync [TICKET]`, `refresh`, `remove TICKET`, `diff`). The library API
mirrors this: `sync()` accepts an optional key, `_add()` is internal, and
`remove()` replaces `remove_root()`. Traversal-config preservation is the
default when no explicit overrides are provided. Standalone `sync` (no ticket)
rebuilds all tracked roots without changing root membership.
