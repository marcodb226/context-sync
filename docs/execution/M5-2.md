    sync/refresh/diff/remove cycle is blocked by an upstream
    `linear-client` bug in the `AttachmentFilter` GraphQL query (field

    library issue, not a context-sync defect ‚Äî the error is correctly
    surfaced as `SystemicRemoteError`.
  - Failure path (invalid ticket): `ContextSync.sync(key='NONEXISTENT-99999')`
    raises `SystemicRemoteError` with message "No issue found for issue
    get by key". Exit: error as expected.
  - Failure path (missing credentials): with all auth env vars cleared,
    `Linear()` raises `LinearConfigurationError("client_id must be
    configured")`. Exit: error as expected.
